Инструкция по запуску:

1. Требования:
Версия Java: Java 23.
Система сборки не требуется (код компилируется и запускается напрямую).
Сторонние библиотеки не используются.

2. Для компиляции программы в терминале выполните:
    javac App.java  
    java App [аргументы]

или:
    java App [аргументы]

3. Аргументы командной строки:
-p <префикс> – добавляет префикс к именам файлов (integers.txt, floats.txt, strings.txt).
-s – вывод статистики (краткий формат).
-f – вывод статистики (полный формат).
-o <путь> – указывает директорию для сохранения файлов.
-a – режим добавления данных в существующие файлы.
<файлы .txt> – список входных файлов с данными.

(если не выбрать ни один префикс статистики (-s/-f), статистика выведена не будет)

Примеры запуска:

    Ввод:
    java App.java -f -p /sample- in1.txt in2.txt -o path/p/a
